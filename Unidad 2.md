## **ESTRUCTURAS CONDICIONALES**
### **Tipos de estructuras condicionales**
#### **1. Simples:**
- Permite ejecutar una acción solamente cuando la condición es verdadera.
  
#### **Pseudocodigo**
```
Algoritmo simple
	
	Si (CONDICION ) Entonces
		instruccion1
		intruccion2
		...
	Fin Si
FinAlgoritmo
```
