**1. Algoritmo**
![Imagen](https://github.com/valentinalalangui05-source/Fundamentos-de-la-Programaci-n./blob/main/Algoritmo.png?raw=true.png)
- Tipos de datos:
1. 🔢*Enteros*: Son valores que no tienen punto decimal, pueden ser negativos o positivos, se incluye el 0
2. 📔*Reales*: Valores que tienen punto decimal, se incluye el 0
3. 🏧*Carácter*: Conformados por un solo carácter, van entre comillas dobles, pueden ser letras, números o símbolos, “x”, “3” etc
4. 🔤*Cadenas*: También conocidos como alfanuméricos, la formación de muchos caracteres forma una cadena
5. 🧐*Lógicos*: Determina si algo es verdadero o falso

- Un algoritmo puede representase también en un lenguaje natural, existen dos herramientas asociadas: 
  - Pseudocódigo
  - Diagrama de flujo
*** 
**2. Pseudocodigo**
   - Es utilizado para entender la estructura de un codigo. Se caracteriza por utilizar una sintaxis flexible y un lenguaje natural. Entre las herramientas mas utlizadas para este proposito podemos mencionar a [Pseint](https://pseint.sourceforge.net/). **Ejemplo**
~~~
Algoritmo Operacion
	
	definir a, b, suma como real;
	
	Escribir ("Ingrasar valor a");
	Leer a;
	
	Escribir ("Ingresar valor b");
	Leer b;
	
	suma= a+b
	
	escribir("El Valor es: "),suma;
	
FinAlgoritmo
~~~
